# 前情提要

- 本项目通过issue评论的方式搜集对导师/任课老师的评价，issue格式请遵循如下格式

  ```c
  自证认识导师、老师：/* 可以描述一些该老师的特征，或用其它你觉得令人信服的方式 */
  个人评价：/* 可从是否点名、课题组经费是否充足、是否提供指导、是否有劣迹等方面评价 */
  ```

- 本项目不对言论真实性负责、言论真实性由读者自行分辨
- 本项目要求所提交issue评论必须遵循格式，不得在issue评论区互相回复掐架，以免给其它读者带来阅读负担
- 如需创建新issue，请遵循其它issue的规范，给新建issue贴上标签
- 不得辱骂、造谣、诽谤，不得发布损害国家利益的内容
